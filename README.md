# ORBIT TV GitHub Pages projekat

Gotova statička web stranica za GitHub Pages.

## Objavljivanje

1. Napravite novi GitHub repository.
2. Uploadujte sve fajlove iz ovog foldera u root repozitorija.
3. Otvorite **Settings → Pages**.
4. Izaberite **Deploy from a branch**, branch **main**, folder **/(root)**.
5. Sačuvajte i otvorite GitHub Pages adresu.

Glavna stranica: `index.html`  
Ugrađeni ORBIT web player: `orbit-web-player.html`  
Converter: `converter.html`
